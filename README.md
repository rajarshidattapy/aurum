# Aurum — AI Mutual Fund Trading Terminal

Aurum is an AI-powered mutual fund trading terminal for Indian markets, combining portfolio analytics, strategy backtesting, and agent-driven decision-making.

## Features
- Portfolio tracking with risk metrics (returns, drawdown, volatility)
- Fund screener with benchmark comparisons
- Strategy backtesting on historical NAV data
- Multi-agent system for portfolio advice, fund analysis, and rebalancing

## Tech Stack
- Backend: Python (FastAPI)
- Frontend: React + TypeScript
- DB: PostgreSQL, Redis
- AI: LLM-based agents
